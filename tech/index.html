Rust

드디어 만난 인생 언어다.
concise 하고 robust 하며 버그나 에러에 깔끔하다.
go 나 java 처럼 사용자의 수준을 고려해 언어를 의도적으로 쉽게 만드는 짓 따위는 하지 않아서 너무너무너무 좋다!
매크로, 제너릭, 포인터, 패턴매칭(!!!), 클로저등등 꼭 필요한게 절묘하게 설계되어있다.
그리고 Cargo and Crates .io 진짜 드림팀은 rust 다.

go 는 rust 를 대충 빠르게 베낀것 같은 느낌.
(속도도 시스템 랭귀지 치고는 좀 너무 느리다.)

rust 는 llvm 위에서 도는 몇 안되는 language 다.
개인적으로 llvm 에서 도는 language 를 좋아하고 문법적으로 인생 언어를 만난 느낌이다.
시스템 엔지니어의 눈높이에 맞는 좋은 랭귀쥐다.
